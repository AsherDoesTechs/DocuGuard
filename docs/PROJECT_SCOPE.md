# DocuGuard Project Scope

## Overview

DocuGuard is an intelligent mobile application that enables users to securely manage digital documents while monitoring their expiration dates.

The application aims to simplify personal document management through automation, organization, and timely reminders.

---

# Included Features

## User Management

- User Registration
- Secure Login
- Logout
- Password Encryption
- Profile Management

---

## Document Management

- Add Document
- Edit Document
- Delete Document
- Upload Files
- Categorize Documents
- Search Documents
- Filter Documents

---

## Dashboard

Displays:

- Total Documents
- Valid Documents
- Expiring Soon
- Expired Documents
- Recent Documents

---

## Expiration Monitoring

Automatic status detection:

- Valid
- Expiring Soon
- Expired

---

## Reminder Notifications

Users receive reminders:

- 30 Days Before Expiration
- 7 Days Before Expiration
- 1 Day Before Expiration
- Expiration Day

---

## User Profile

- Update Profile
- Change Password
- Notification Preferences

---

# Out of Scope

The following features are not included in Version 1.0:

- OCR Document Scanning
- AI Document Classification
- Face Recognition
- Multi-user Collaboration
- Desktop Version
- Offline Synchronization
- Digital Signature Support

These may be considered for future releases.

---

# Target Users

- Students
- Professionals
- Employees
- Business Owners
- Freelancers
- Individuals managing personal records

---

# Success Criteria

The project is considered successful if users can:

- Register an account.
- Securely log in.
- Upload documents.
- Organize documents.
- Receive expiration reminders.
- Quickly retrieve stored documents.
